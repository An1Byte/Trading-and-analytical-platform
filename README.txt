Инструкция по установке программы - коннектора:

1. Разархивировать архив.
2. Нажать на INSTALL.exe и установить программу - коннектор и nodeJS сервер если у вас нет последнего.
	Возможны варианты:
		а. Установка программы - коннектора без установки nodeJS сервера.
		б. Установка программы - коннектора вместе с nodeJS сервером.
3. После установки nodeJS и программы вы можете проверить есть ли у вас node js и виден ли он в глобальной области. Для этого вызовите консоль "Win + R", в появившемся поле напишите "cmd". В появившемся консольном окне наберите следующую команду и нажмите enter: "node -v" без указания кавычек. Если все было установленно правильно, то в ответ вам выдадут строчку с названием текущей версии node js. На этом консоль можно закрыть.
4. В папке Experts есть соответствующие терминалам эксперты, которые нужно поместить в пространство вашего терминала. О том, как это сделать существуют множество туториалов.
5. Включите терминал (терминалы) и перетащите эксперт на интересующие вас инструменты, которые вы хотите видеть в коннекторе. При перетаскивании вам предложат заполнить поле(по умолчанию = 1000) - это поле является значением подкачиваемой в программу-коннектор исторических данных по инструменту.
6. В папке InServer есть файл exe, который называется InvizzzApplication.exe. Нажимая на него, откроется окно в котором вы должны прописать абсолютные пути к папке Invizzz, в пространствах ваших терминалов. Эту папку создают скрипты. После того, как укажите пути к папкам жмете на зеленую кнопку и откроется браузер chrome. Программа ориентированна именно на этот браузер, поэтому его также следует установить, у кого его нет !


* При закрытии браузерной вкладки сервер будет автоматически отключен и чтобы запустить приложение снова, переходим к пункту 6, правда теперь указанные пути уже будут присутствовать в полях, просто жмем зеленую кнопку.
* Программа использует только 1 порт 8080, по этому создавать множество вкладок ссылаясь на данный порт категорически запрещено. Если вы случайно создали дубль своего приложения, то для корректной работы лучше все закрыть и запустить приложение заного.
